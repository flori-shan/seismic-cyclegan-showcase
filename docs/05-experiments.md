## 3. 主要发现

### 3.1 Overlap 切块（E1）— **最重要结论**

**现象（无 overlap）**：

- patch 边缘道生成振幅偏浅  
- 拼接后出现**明显缝隙 / 错位**  
- 大图视觉上「一块一块」不连续

**改进（有 overlap）**：

- 边缘区域被相邻 patch 覆盖  
- 拼接更平滑，同相轴连续性更好

> 面试话术：「这不是调参能解决的，是**推理工程**问题 — 训练在 patch 上，部署必须考虑重叠融合。」

### 3.2 振幅归一化（E4）

**现象**：预测图整体偏深或偏浅，与真值对比色差大。

**原因**：域 X 缺失道后能量分布与域 Y 不同。

**处理**：训练前两域统一归一化策略；推理后 inverse transform。

### 3.3 Epoch 对比（E2）

- 40 epoch：部分细节未收敛，高频略糊  
- 200 epoch：纹理更细，但需警惕过拟合与训练时间  
- 与论文 200 epoch 设定对标

### 3.4 生成器对比（E3）

| 生成器 | 观察 |
|--------|------|
| U-Net | 跳跃连接利于保留浅层同相轴，**默认首选** |
| ResNet | 全局一致性好，细节略逊，作对照 |

### 3.5 优化器（E6）

- **Mac MPS + Adam**：训练崩溃 / NaN  
- 切换 **Adagrad** 后稳定  
- 体现**跨平台深度学习工程**经验
