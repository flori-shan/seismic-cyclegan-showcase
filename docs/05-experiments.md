## E1 · overlap 有无 — **影响最大**

![overlap 对比](../assets/images/05-overlap-stitch.png)

| | 无 overlap | 有 overlap |
|---|-----------|------------|
| **现象** | patch 边缘道振幅偏浅；拼起来有竖缝 | 重叠区融合，同相轴连续 |
| **原因** | 卷积边界 + 小块缺少上下文 | 相邻 patch 互相覆盖 |
| **结论** | 推理阶段必须做 overlap，不是调参能救 |

要点：训练在 patch 上做，部署必须考虑重叠融合。

## E2 · 振幅归一化

![振幅对齐](../assets/images/06-amplitude-align.png)

| | 未对齐 | 对齐后 |
|---|--------|--------|
| **现象** | 预测图整体偏浅/偏深 | 颜色深度接近真值 |
| **原因** | 域 X 缺道后能量分布 ≠ 域 Y | 两域统一 min-max / z-score |
| **结论** | 预处理是隐性关键步骤 |

## 其他发现

- **40 vs 200 epoch**：200 纹理更细，但训练更久，需防过拟合  
- **U-Net vs ResNet**：U-Net 跳跃连接保留浅层同相轴，作默认；ResNet 作对照  
- **波形数少的 patch**：特征分散，效果不如波纹多的样本（文档记录约 15 vs 21 道波纹）
