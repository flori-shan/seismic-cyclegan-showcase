# 07 · 工程踩坑与解决

> 图：[overlap 拼接](../assets/images/05-overlap-stitch.png) · [振幅对齐](../assets/images/06-amplitude-align.png) · [实验迭代](../assets/images/09-experiment-timeline.png)

> 这部分集中记录了项目落地过程中的真实工程经验。

---

## 1. Mac MPS + Adam 训练崩溃

| 项目 | 内容 |
|------|------|
| **现象** | Apple Silicon 上使用 MPS 后端，Adam 优化器训练中出现 NaN / 崩溃 |
| **原因** | MPS 对部分 TF 算子与 Adam 二阶矩估计兼容性不完善 |
| **解决** | 改用 Adagrad / SGD / RMSProp；或降级 CPU 验证 |
| **收获** | 跨平台部署需准备**优化器备选方案**，不能假设论文默认配置即开箱即用 |

---

## 2. 预测图空白 / 全黑

| 项目 | 内容 |
|------|------|
| **现象** | 推理输出全黑或近乎无纹理 |
| **排查** | ① 归一化未反变换 ② 激活函数输出范围 ③ 权重未正确加载 ④ 输入通道顺序错误 |
| **解决** | 检查 inference 脚本中 `tanh` → `[0,1]` 映射；确认 checkpoint 与模型结构一致 |
| **收获** | 生成模型 debug 要**从数据流逐层打印 min/max** |

---

## 3. 颜色深度不一致（过深 / 过浅）

| 项目 | 内容 |
|------|------|
| **现象** | 生成炮集与真值对比，整体偏深或偏浅 |
| **原因** | 域 X（缺失道）与域 Y 振幅统计量不同，GAN 学到的是「分布」而非绝对振幅 |
| **解决** | 训练前两域**统一归一化**；必要时加 identity loss 权重 |
| **收获** | 无配对翻译中，**预处理对齐两域分布**是隐性关键步骤 |

---

## 4. 测试集 shape 不一致

| 项目 | 内容 |
|------|------|
| **现象** | `batch` 时报错：各样本 H×W 不同 |
| **原因** | 炮集切块尺寸不统一 |
| **解决** | `resize` 到固定尺寸；或 `padded_batch`；或按尺寸分桶 |
| **收获** | 工业数据很少像 ImageNet 一样整齐，**数据管道鲁棒性**优先 |

---

## 5. 预测结果乱序 / 拼接错位

| 项目 | 内容 |
|------|------|
| **现象** | 拼接后大图道顺序错乱，同相轴斜率断裂 |
| **原因** | `tf.data` 默认 `shuffle=True` 用于训练，推理时未关闭 |
| **解决** | 推理 pipeline：`shuffle=False`，按**文件名排序**后逐张预测 |
| **收获** | **训练与推理数据管道要分开配置**，shuffle 是常见隐蔽 bug |

---

## 6. Patch 边缘道质量差 + 拼接缝隙

| 项目 | 内容 |
|------|------|
| **现象** | 每个 patch 左右边缘道振幅偏浅；拼接有明显竖缝 |
| **原因** | 卷积边界效应 + 训练 patch 无上下文 |
| **解决** | **overlap 切块** + 拼接时取中心区域或加权融合 |
| **收获** | 大图像生成任务的**经典工程问题**，论文往往不写推理细节 |

---

## 7. 生成器结构选择困惑

| 项目 | 内容 |
|------|------|
| **现象** | U-Net 与 ResNet 结果风格不同，难以取舍 |
| **处理** | 固定其他变量，做 A/B 对比；U-Net 保留浅层细节更适合同相轴 |
| **收获** | 不要迷信「论文默认」— **领域数据特性**决定结构优劣 |

---

## 8. 踩坑总结表（速查）

| # | 问题 | 根因 | 修复 |
|---|------|------|------|
| 1 | Adam 崩溃 | MPS 兼容 | 换优化器 |
| 2 | 空图 | 归一化/权重 | 数据流 debug |
| 3 | 色差 | 域分布不一致 | 振幅对齐 |
| 4 | batch 报错 | 尺寸不一 | resize / pad |
| 5 | 拼接错位 | shuffle | 排序推理 |
| 6 | 边缘缝隙 | 边界效应 | overlap |
